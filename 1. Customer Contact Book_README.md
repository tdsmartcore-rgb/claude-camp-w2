# 顾客名册管理器

Python 字典（Dictionary）入门练习项目。

## 功能

- 添加顾客（姓名、邮箱、电话）
- 按姓名关键字查询（支持模糊匹配）
- 按编号删除顾客
- 输入异常时给出友好提示，不会崩溃

## 运行方法

确保已安装 Python，然后在终端执行：

```
python customer_manager.py
```

## 使用到的 Python 知识点

| 知识点 | 用途 |
|--------|------|
| Dictionary | 存储顾客信息 |
| `.items()` | 遍历名册 |
| `del` | 删除顾客 |
| `in` | 判断编号是否存在 / 关键字匹配 |
| `while` 循环 | 持续显示菜单 |
| `try / except` | 捕获异常，防止崩溃 |
