# 文本词频统计器

Python 入门练习项目，输入一段英文文字，统计每个单词出现的次数。

## 功能

- 输入一段英文文字，输出每个单词的出现次数
- 用字典（Dictionary）存储统计结果
- 按出现次数从高到低排序输出
- 忽略大小写差异（Hello 和 hello 算同一个词）

## 运行方法

确保已安装 Python，然后在终端执行：

```
python word_counter.py
```

## 使用示例

输入：
```
hello world hello Python python WORLD
```

输出：
```
共找到 6 个单词（含重复）

── 词频统计结果 ──────────────────
单词            出现次数
─────────────────────────
hello           2 次
world           2 次
python          2 次
```

## 练习到的 Python 知识点

| 知识点 | 用途 |
|--------|------|
| Dictionary | 存储每个单词和对应的次数 |
| `if / elif / else` | 判断单词是否已在字典里 |
| `try / except` | 捕获错误，防止程序崩溃 |
| `pass` | 跳过空字符串，不做任何处理 |
| `.lower()` | 把文字全部转成小写 |
| `.split()` | 把一段话拆分成单词列表 |
| `sorted()` | 按次数从高到低排序 |

## 错误处理说明

| 错误类型 | 触发情景 | 处理方式 |
|----------|---------|---------|
| `ValueError` | 输入内容不合法 | 提示用户重新运行 |
| `NameError` | 变量不存在或拼写错误 | 提示用户重新运行 |
| `SyntaxError` | 代码写法错误 | 运行前检查代码，无法用 try 捕获 |
