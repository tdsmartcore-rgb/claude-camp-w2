# 📝 Python To-Do List

一个用 Python 写的命令行待办事项小程序，支持本地文件保存，适合 Python 初学者学习实际项目。

---

## Funtion

- ✅ Add to do items
- ✅ Mark the task as done
- ✅ View current task list
- ✅ Data saved in `todos.json`，ensure no data loss after restart.

---

## 运行方式

**第一步：进入脚本所在目录**

```bash
cd /你的文件夹路径
```

**第二步：运行程序**

```bash
python todo_list.py
```

> ⚠️ 不要在根目录 `/` 下直接运行，会因为没有写入权限而报错。

---

## 使用说明

程序启动后，按菜单输入数字进行操作：

```
请选择操作：
  1. 查看清单
  2. 添加待办
  3. 完成待办
  0. 退出程序
```

数据会自动保存到同目录下的 `todos.json` 文件，无需手动操作。

---

## 文件结构

```
你的文件夹/
├── todo_list.py    # 主程序
└── todos.json      # 自动生成，存储待办数据（第一次运行后出现）
```

---

## 涉及的 Python 知识点

| 知识点 | 在本项目中的用途 |
|---|---|
| `dictionary` | 每条待办事项的数据结构（task + done） |
| `try...except` | 处理文件不存在、输入非数字、编号越界等异常 |
| `pass` | 用户直接按回车时跳过，不报错 |
| `FileNotFoundError` | 程序第一次运行时文件还不存在的情况 |
| `ValueError` | 用户输入了文字而不是数字时的情况 |
| `IndexError` | 用户输入了超出范围的编号时的情况 |

---

## 环境要求

- Python 3.8 及以上
- 无需安装任何第三方库，全部使用标准库（`json`、`os`）
