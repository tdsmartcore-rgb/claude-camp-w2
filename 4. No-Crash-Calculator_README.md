# 安全的计算器 🧮

一个不会崩溃的命令行计算器，适合 Python 初学者练手。

## 功能

- 支持加、减、乘、除四则运算
- 输入除数为零时，给出提示而不是崩溃
- 输入非数字时，给出提示而不是崩溃
- 输入 `quit` 优雅退出

## 使用方法

```bash
python calculator.py
```

按照提示依次输入第一个数字、运算符、第二个数字，即可得到结果。

```
欢迎使用安全计算器！输入 quit 退出。
支持的运算符：+ - * /

请输入第一个数字（或 quit 退出）：10
请输入运算符（+ - * /）：/
请输入第二个数字：2
✅ 10.0 / 2.0 = 5.0
```

## 错误处理示例

```
请输入第一个数字（或 quit 退出）：abc
❌ 请输入有效的数字！

请输入第二个数字：0
❌ 除数不能为零！
```

## 涉及的 Python 知识点

| 知识点 | 用途 |
|--------|------|
| `dict` + `lambda` | 把运算符映射到对应的计算函数 |
| `try / except` | 捕获错误，防止程序崩溃 |
| `while True` + `break` | 持续运行，输入 quit 时退出 |
| `continue` | 出错后跳回循环顶部重新开始 |

## 环境要求

- Python 3.x
- 无需安装任何第三方库
