# Python Week 2 Practice Projects

A collection of beginner Python exercises. These four small programs cover the practical use of Dictionary and exception handling (try/except) from scratch.

---

## What You'll Learn

These four projects cover the following core concepts:

| Concept | What You Learn |
|---------|---------------|
| `Dictionary` | Store and retrieve data using key-value pairs |
| `try / except` | Catch errors and write programs that don't crash |
| `while` loop | Keep a program running until the user decides to quit |
| `lambda` | Write a function in one line, used together with dictionaries |
| File I/O | Save data to a local JSON file so it persists after restart |

---

## Projects

### 1. Customer Contact Book

Manage customer names, emails, and phone numbers — with support for adding, searching, and deleting records.

**Problem it solves:** How to use a dictionary to store and manage a set of structured data.

```bash
python customer_manager.py
```

---

### 2. Word Frequency Counter

Enter a passage of English text and get each word's frequency, sorted from most to least common.

**Problem it solves:** How to use a dictionary for counting, and how to clean and process text input.

```bash
python word_counter.py
```

---

### 3. Command-Line To-Do List

Add, complete, and view to-do items. Data is saved to a local JSON file and survives restarts.

**Problem it solves:** How to persist dictionary data to a file so the program remembers its state after closing.

```bash
python todo_list.py
```

---

### 4. No-Crash Calculator

A four-operation calculator that handles invalid input and division by zero gracefully — no crashes.

**Problem it solves:** How to replace long if-elif chains with a dictionary, and how to handle all kinds of bad user input.

```bash
python calculator.py
```

---

## Requirements

- Python 3.8 or above
- No third-party libraries required
